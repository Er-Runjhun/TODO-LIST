# 📝 Django To-Do List Application

A simple and user-friendly To-Do List web application built using Django. This application allows users to create, manage, update, and delete tasks efficiently, helping them stay organized and productive.

## 🚀 Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete completed or unwanted tasks
- Mark tasks as completed
- Simple and responsive user interface
- Django-powered backend

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite3
